
# The OpenAI Model Spec

The Model Spec specifies desired behavior for the models underlying OpenAI's products (including our APIs).

See the latest Model Spec rendered for human reading at [model-spec.openai.com](https://raw.githubusercontent.com/openai/model_spec/main/model-spec.openai.com),
read more about the Model Spec on the [OpenAI Blog](https://raw.githubusercontent.com/openai/model_spec/main/OpenAI Blog), or
check out the latest [Release Notes](https://raw.githubusercontent.com/openai/model_spec/main/Release Notes).

This repository contains the [markdown source](https://raw.githubusercontent.com/openai/model_spec/main/markdown source) for the Model Spec and an archive of all released
HTML versions of the Model Spec (starting from the second release on 2025-02-12).
