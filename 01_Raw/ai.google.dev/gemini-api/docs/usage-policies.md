# رصد إساءة الاستخدام

تلتزم Google بتطوير الذكاء الاصطناعي واستخدامه بشكلٍ مسؤول. ولضمان سلامة Gemini API وسلامة بياناته، وضعنا إرشادات السياسة هذه.
باستخدام Gemini API، أنت توافق على الإرشادات التالية وبنود الخدمة الإضافية الخاصة بـ [Gemini API
وسياسة الاستخدام المحظور للذكاء الاصطناعي التوليدي [المحظور](https://policies.google.com/terms/generative-ai/use-policy?hl=ar).](https://ai.google.dev/gemini-api/terms?hl=ar)

## كيفية رصد حالات إساءة الاستخدام

يستخدم فريق الأمان والثقة في Google مزيجًا من العمليات الآلية واليدوية لرصد حالات إساءة الاستخدام المحتملة لـ Gemini API وGoogle AI Studio من أجل تطبيق سياساتنا.

- **الرصد الآلي:** تفحص الأنظمة الآلية استخدام واجهة برمجة التطبيقات بحثًا عن انتهاكات لسياسة الاستخدام المحظور، مثل كلام يحض على الكراهية والمضايقة والمحتوى الجنسي الفاضح والمحتوى الخطير.
- **الرصد اليدوي:** إذا كان أحد المشاريع يعرض باستمرار نشاطًا مشبوهًا، قد يتم وضع علامة عليه لمراجعته يدويًا من قِبل موظفين Google المفوّضين.

## كيفية معالجة البيانات

تحتفظ Google بالبيانات التالية لمدة 55 يومًا لأغراض رصد انتهاكات سياسة الاستخدام المحظور ومنعها، وذلك للحفاظ على سلامة الخدمات وأمانها، وأي إفصاحات قانونية أو تنظيمية مطلوبة:

- **الطلبات:** طلبات النص التي ترسلها إلى واجهة برمجة التطبيقات
- **المعلومات السياقية:** أي سياق إضافي تقدّمه مع طلباتك
- **الناتج:** الردود التي تم إنشاؤها بواسطة Gemini API

## كيفية التحقيق في المشاكل المحتملة

عندما يتم وضع علامة على الطلبات أو نواتج النماذج من قِبل فلاتر الأمان وأنظمة رصد إساءة الاستخدام الموضّحة أعلاه، يجوز لموظفي Google المفوّضين تقييم المحتوى الذي تم وضع علامة عليه، وتأكيد التصنيف أو التحديد أو تصحيحهما استنادًا إلى الإرشادات والسياسات المحدّدة مسبقًا. لا يمكن للموظفين المفوّضين في Google الوصول إلى البيانات لمراجعتها يدويًا إلا من خلال منصة داخلية لتقييم الحوكمة وإدارة المراجعة. عند تسجيل البيانات لرصد إساءة الاستخدام، لا يتم استخدامها إلا لغرض تطبيق السياسة ومنع انتهاكاتها.
ولا يتم استخدامها لتدريب أي نماذج ذكاء اصطناعي أو تعلُّم آلة أو تحسينها، باستثناء النماذج المستخدَمة خصيصًا لتطبيق السياسة.

## التعاون معك لضمان الامتثال للسياسة

إذا كان استخدامك لـ Gemini لا يتوافق مع سياساتنا، قد نتّخذ الخطوات التالية:

- **التواصل معك:** قد نتواصل معك عبر البريد الإلكتروني لفهم حالة استخدامك واستكشاف طرق لجعل استخدامك متوافقًا مع السياسات.
- **سقف مؤقت للاستخدام:** قد نحدّ من إمكانية وصولك إلى Gemini API.
- **تعليق مؤقت:** قد نعلّق مؤقتًا إمكانية وصولك إلى Gemini API.
- **إغلاق الحساب:** كملاذ أخير وفي حالات الانتهاكات الخطيرة، قد نوقف نهائيًا إمكانية وصولك إلى Gemini API وخدمات Google الأخرى.

إذا تواصلنا معك بشأن تعليق الحساب أو إغلاقه، سنقدّم لك أيضًا رابطًا يمكنك من خلاله تقديم طلب إعادة نظر.
