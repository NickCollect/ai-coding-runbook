# Menyesuaikan dengan Gemini API

Dengan penghentian Gemini 1.5 Flash-001 pada Mei 2025, kami tidak lagi memiliki
model yang mendukung penyesuaian di Gemini API atau AI Studio, tetapi
model ini didukung di [Platform Agen Gemini Enterprise](https://cloud.google.com/gemini-enterprise-agent-platform/models/gemini-use-supervised-tuning?hl=id).

Kami belum memiliki rencana untuk segera menghadirkan dukungan penyesuaian, tetapi kami ingin [mendengar
pendapat Anda di forum developer kami](https://discuss.ai.google.dev/c/gemini-api/4?hl=id)
jika penyesuaian penting untuk kasus penggunaan Anda.
