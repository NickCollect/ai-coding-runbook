# ตัวอย่าง Antigravity

เอเจนต์ที่มีการจัดการแบบอเนกประสงค์ที่ทรงพลังซึ่งออกแบบมาสำหรับเวิร์กโฟลว์แบบหลายขั้นตอนที่ทำงานโดยอัตโนมัติ โมเดลนี้วางแผน ให้เหตุผล เรียกใช้โค้ด จัดการไฟล์ และค้นหา
