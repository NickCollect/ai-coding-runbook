# Imagen 4

這款高效能引擎可進行進階視覺合成，提供精密的創意控制功能，並輸出逼真的圖像。Imagen 4 適合用於專業品牌宣傳、複雜場景構圖，以及需要精確文字算繪和複雜光影的高保真設計工作。

[在 Google AI Studio 中試用](https://aistudio.google.com?model=imagen-4.0-generate-001&hl=zh-tw)
