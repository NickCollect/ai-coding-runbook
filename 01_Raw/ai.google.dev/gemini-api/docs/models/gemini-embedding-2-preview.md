# Gemini Embedding 2 预览版模型

我们的首个多模态嵌入模型，可将文本、图片、视频、音频和 PDF 高效地映射到单个统一的嵌入空间。Gemini Embedding 2 模型最适合跨模态语义搜索、文档检索和推荐系统，这些应用需要在大型多模态数据集中快速进行可扩缩的相似度计算。
