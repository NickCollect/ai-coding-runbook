# Lyria RealTime 實驗功能

這項實驗性引擎可合成高傳真度的音樂，並提供創新功能，用於生成及轉換音訊內容。Lyria RealTime 最適合用於 AI 輔助的詞曲創作、生成獨特的樂器伴奏，以及需要深入控制旋律和節奏的創意音訊工作流程，且不含人聲。
