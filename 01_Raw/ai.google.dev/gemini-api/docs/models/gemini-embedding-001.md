# Gemini Embedding 模型

這款專用引擎可提供高維度向量表示法，有效率地將文字和圖片對應至數值。Gemini Embedding 模型最適合語意搜尋、文件檢索和推薦系統，這類系統需要對大型資料集進行快速、可擴充的相似度計算。
