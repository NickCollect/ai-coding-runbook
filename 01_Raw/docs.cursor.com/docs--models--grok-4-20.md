Grok 4.20 is xAI's coding-focused model. It supports thinking mode, a 200k default context window, and up to 2M tokens in long-context mode.

## Strengths

- Up to 2M tokens of context. Useful for large repositories, long transcripts, and multi-file tasks.
- Supports thinking mode for harder coding and debugging work.
- Fast enough for iterative agent workflows in Cursor.

## Pricing

Cursor [plans](https://cursor.com/docs/models-and-pricing.md) include two usage pools. Grok 4.20 draws from the **API** pool, which charges at the rates below. Base pricing is $2.00 per 1M input tokens, $0.20 per 1M cached tokens, and $6.00 per 1M output tokens through 200k context. Above 200k context, those rates double. All prices are per million tokens.
