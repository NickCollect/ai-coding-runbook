Gemini 3 Flash is Google's speed-optimized model. At $0.50 per million input tokens, it's one of the cheapest options available. Use it for simple coding tasks, quick edits, and executing plans written by a more capable model.

## Strengths

- One of the fastest models. Optimized for low-latency responses. Good for simple edits, quick tasks, and subagent work where speed is critical.
- $0.50/1M input tokens. One of the most affordable options. Strong for cost-constrained workflows.
- Supports reasoning mode for tasks that need it, while keeping costs low.
