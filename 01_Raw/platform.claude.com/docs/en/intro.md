# Intro to Claude

Claude is a highly performant, trustworthy, and intelligent AI platform built by Anthropic. Claude excels at tasks involving language, reasoning, analysis, coding, and more.

---

<Tip>

The latest generation of Claude models:

**Claude Opus 4.7** - Our most capable model for complex reasoning and agentic coding, with a step-change jump over Claude Opus 4.6. [Learn more](https://platform.claude.com/docs/en/Learn more).

**Claude Sonnet 4.6** - Frontier intelligence at scale—built for coding, agents, and enterprise workflows. [Learn more](https://platform.claude.com/docs/en/Learn more).

**Claude Haiku 4.5** - Fastest model with near-frontier intelligence. [Learn more](https://platform.claude.com/docs/en/Learn more).

</Tip>

<Note>
Looking to chat with Claude? Visit [claude.ai](https://platform.claude.com/docs/en/claude.ai).
</Note>

Anthropic offers two ways to build with Claude, each suited to different use cases:

| | Messages API | Claude Managed Agents |
|---|---|---|
| **What it is** | Direct model prompting access | Pre-built, configurable agent harness that runs in managed infrastructure |
| **Best for** | Custom agent loops and fine-grained control | Long-running tasks and asynchronous work |
| **Learn more** | [Messages API docs](https://platform.claude.com/docs/en/Messages API docs) | [Claude Managed Agents docs](https://platform.claude.com/docs/en/Claude Managed Agents docs) |

## Recommended path for new developers

Follow these steps to go from zero to a working Claude integration.

<Steps>
  <Step title="Make your first API call">
    Set up your environment, install an SDK, and send your first message to Claude.

    [Go to the quickstart](https://platform.claude.com/docs/en/Go to the quickstart)
  </Step>
  <Step title="Understand the Messages API">
    Learn the core request and response structure, including multi-turn conversations, system prompts, and stop reasons.

    [Read the Messages API guide](https://platform.claude.com/docs/en/Read the Messages API guide)
  </Step>
  <Step title="Choose the right model">
    Compare Claude models by capability and cost to pick the best fit for your use case.

    [See the models overview](https://platform.claude.com/docs/en/See the models overview)
  </Step>
  <Step title="Explore features and tools">
    Discover what Claude can do: extended thinking, web search, file handling, structured outputs, and more.

    [Browse the features overview](https://platform.claude.com/docs/en/Browse the features overview)
  </Step>
</Steps>
