Weekly digest covering Claude Code v2.1.83 → v2.1.85 (March 23–27, 2026). Six features highlighted.

**Auto mode (research preview)**: hands permission prompts to a classifier — safe edits/commands run, destructive/suspicious blocked and surfaced. Middle ground between approving every write and `--dangerously-skip-permissions`. Cycle to it with Shift+Tab or set as default:
```json
{ "permissions": { "defaultMode": "auto" } }
```

**Computer use (Desktop)**: Claude controls actual desktop in Claude Code Desktop — open native apps, click iOS Simulator, drive hardware control panels, GUI verification. Off by default; asks before each action. Best for things nothing else can reach (apps without API, proprietary tools, GUI-only workflows).

**PR auto-fix (Web)**: toggle on PR open, walk away. Claude watches CI, fixes failures, handles nits, pushes until green. Available in claude.ai/code CI panel.

**Transcript search (v2.1.83)**: press `/` in transcript mode (Ctrl+O) to search; `n`/`N` step forward/back through matches.

**PowerShell tool (preview, v2.1.84)**: native PowerShell on Windows alongside Bash — cmdlets, object pipelines, native paths. Opt in via `CLAUDE_CODE_USE_POWERSHELL_TOOL=1`.

**Conditional hooks (v2.1.85)**: hooks can declare `if` field using permission-rule syntax to scope when they fire. E.g. pre-commit hook only on `Bash(git commit *)`:
```json
"hooks": {
  "PreToolUse": [{
    "hooks": [{
      "if": "Bash(git commit *)",
      "type": "command",
      "command": ".claude/hooks/lint-staged.sh"
    }]
  }]
}
```

**Other wins** (compact list):
- Plugin `userConfig` now public — prompt for settings at enable-time, keychain-backed secrets
- Pasted images insert `[Image #N]` chips for positional reference
- `managed-settings.d/` drop-in directory for layered policy
- New hook events: `CwdChanged`, `FileChanged` (direnv-style use cases)
- Agents can declare `initialPrompt` in frontmatter to auto-submit a first turn
- `Ctrl+X Ctrl+E` opens external editor (readline-compatible)
- Interrupting before any response restores your input automatically
- `/status` works while Claude is responding
- Deep links open in preferred terminal
- Idle-return nudge to `/clear` after 75+ min away
- VS Code: rate-limit banner, Esc-twice rewind picker
