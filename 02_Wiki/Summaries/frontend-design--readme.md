Plugin (skill-style) that auto-engages on frontend work. Generates "distinctive, production-grade frontend interfaces that avoid generic AI aesthetics" — bold aesthetic choices, distinctive typography + color palettes, high-impact animations, context-aware implementation.

Trigger automatically (Claude decides when relevant): "Create a dashboard for a music streaming app", "Build a landing page for an AI security startup", "Design a settings panel with dark mode".

References the [Frontend Aesthetics Cookbook](https://github.com/anthropics/claude-cookbooks/blob/main/coding/prompting_for_frontend_aesthetics.ipynb) for prompting guidance.

Authors: Prithvi Rajasekaran, Alexander Bricken (both @anthropic.com). Brief 31-line file.
