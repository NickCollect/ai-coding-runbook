Weekly digest covering Claude Code v2.1.86 → v2.1.91. Five highlighted features:

1. **Computer use in the CLI** (research preview): Claude opens native apps, clicks UI, tests changes, fixes them — all from terminal. Prior week landed in Desktop; now in CLI for native iOS/macOS/GUI-only apps without APIs. Enable via `/mcp` → toggle `computer-use`. Example: "Open the iOS simulator, tap through onboarding, and screenshot each step".

2. **`/powerup`** (v2.1.90): interactive in-product lessons with animated demos teaching CC features.

3. **Flicker-free rendering** (v2.1.89): new alt-screen renderer with virtualized scrollback. Prompt input pinned bottom; mouse selection works across long convos; no flicker on redraw. Opt-out: unset `CLAUDE_CODE_NO_FLICKER`.

4. **MCP result-size override** (v2.1.91): MCP server authors can raise truncation cap on a specific tool via `anthropic/maxResultSizeChars` in `tools/list` entry's `_meta`, up to **500K char hard ceiling**. Previously global cap → large payloads (DB schemas, file trees) hit default and got persisted to disk with file ref.

5. **Plugin executables on PATH** (v2.1.91): `bin/` directory at plugin root → CC adds it to Bash tool's PATH while plugin enabled. Bare command invocation, no absolute path needed.

**Other wins**:
- Auto mode follow-ups: `PermissionDenied` hook on classifier denials (return `retry: true` for Claude to try alternative); `/permissions` → Recent → `r` to retry manually.
- New `defer` value for `permissionDecision` in `PreToolUse` hooks: `-p` sessions pause at tool call, exit with `deferred_tool_use` payload, resumed with `--resume`.
- `/buddy`: small creature watches you code (April 1st joke).
- `disableSkillShellExecution` setting blocks inline shell from skills/slash commands/plugin commands.
- Edit tool now works on files viewed via `cat` or `sed -n` without separate Read.
- Hook output > 50K saved to disk with path + preview instead of injected into context.
- Thinking summaries off by default in interactive (`showThinkingSummaries: true` to restore).
- Voice mode: push-to-talk modifier combos, Windows WebSocket, macOS Apple Silicon mic permission.
- `claude-cli://` deep links accept multi-line prompts (encoded `%0A`).
