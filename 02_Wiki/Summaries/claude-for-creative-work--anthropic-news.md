Apr 28, 2026 — **Claude for Creative Work** announced — a set of MCP-based connectors letting Claude work alongside the software creative pros use.

**New connectors.** Ableton (Live and Push docs), Adobe for creativity (50+ Creative Cloud apps including Photoshop, Premiere, Express), Affinity by Canva (batch image adjustments, layer renaming, file export, custom-feature generation), Autodesk Fusion (3D modeling), Blender (NL interface to Python API), Resolume Arena and Wire (live VJ/AV control), SketchUp (NL → 3D modeling starter), Splice (royalty-free sample search).

**Use cases highlighted.**
- On-demand tutor for complex creative software.
- Extending tools with code (custom shaders, procedural animation, parametric models via Claude Code).
- Pipeline bridging (translate formats, restructure data, sync assets across design/3D/audio).
- Rapid exploration / handoff via Claude Design.
- Repetitive production work (batch processing, project scaffolding, procedural changes).

Positioning: AI doesn't replace taste/imagination but expands what's possible — faster ideation, larger projects, less manual toil. Integration into existing creative workflows is the unlock.
