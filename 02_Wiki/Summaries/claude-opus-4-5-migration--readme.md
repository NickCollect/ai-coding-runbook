Skill plugin that automates migrating code and prompts from Sonnet 4.x / Opus 4.1 to Opus 4.5: handles model string updates, beta headers, configuration details. Continue using to adjust prompts post-migration if issues arise.

Trigger: "Migrate my codebase to Opus 4.5".

Author: William Hu (whu@anthropic.com). Brief 22-line file referencing the [Claude 4 prompting best practices](https://platform.claude.com/docs/en/build-with-claude/prompt-engineering/claude-4-best-practices) for further guidance.
