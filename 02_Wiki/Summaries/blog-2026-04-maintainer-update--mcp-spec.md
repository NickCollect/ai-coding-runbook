David Soria Parra announces two leadership changes:

**New Core Maintainer: Clare Liguori** — Senior Principal Engineer at AWS working on agentic AI developer tooling (Kiro, Strands Agents SDK). Decade+ at AWS across Proton, ECS, Code Suite, OSS. Brings depth in developer-tooling and agent-runtime perspective; involved in unsolicited tasks discussion (PR 2229), agent execution model, and Triggers & Events working group.

**Den Delimarsky promoted to Lead Maintainer** (joining David Soria Parra) — Member of Technical Staff at Anthropic working across spec, SDKs, governance, DX. Previously Principal Product Engineer at Microsoft CoreAI. Most visible work in authorization and security: co-authored the authorization spec, brought RFC 8707 Resource Indicators to the spec (PR 734), continued via SEP-835, SEP-1024, SEP-2350 family. Led the 2025-11-25 spec release. Co-leads the Security Interest Group. Built the MCP repo data tracker for SDK and spec maintainers.

**Context**: governance model from summer 2025 has held up through two spec releases and the move to AAIF, plus growing SEP volume. These additions give the project the leadership capacity for the next phase.
