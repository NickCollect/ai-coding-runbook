Nov 18, 2025 — **Claude available in Microsoft Foundry and expanded Microsoft 365 Copilot integration**, part of expanded Microsoft-Anthropic partnership.

**Microsoft Foundry.** Sonnet 4.5, Haiku 4.5, Opus 4.1 now in public preview. Serverless deployment (Anthropic manages infrastructure). Eligible for Microsoft Azure Consumption Commitment (MACC); works with existing Azure agreements/billing — no separate vendor contracts. SDKs in Python, TypeScript, C# with Microsoft Entra auth. Global Standard deployment rolling out at standard API pricing; US DataZone coming.

**Microsoft 365 Copilot.** Claude already powers the **Researcher agent** (complex multi-step research) and enables custom agent development in Copilot Studio. **Excel Agent Mode** now offers Claude in preview to build/edit spreadsheets — generate formulas, analyze data, identify errors, iterate.

**Model positioning in Foundry.**
- Sonnet 4.5 — best coding model + strongest agent model.
- Haiku 4.5 — fastest, near-frontier at one-third Sonnet cost; sub-agents, customer support, content moderation, real-time coding.

Removes procurement overhead (separate vendor contracts and billing) for enterprises invested in Microsoft ecosystem. Established Foundry as one of the three major cloud channels alongside Bedrock and Vertex AI.
