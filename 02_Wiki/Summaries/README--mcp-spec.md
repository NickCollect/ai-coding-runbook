Brief landing page for the canonical `modelcontextprotocol/modelcontextprotocol` repository, which contains:

1. **The MCP specification** itself (under `docs/specification/`)
2. **The MCP protocol schema** — TypeScript-first source of truth at `schema/2025-11-25/schema.ts`, also published as JSON Schema (`schema.json`) for wider tooling compatibility
3. **The official MCP documentation** built with Mintlify and published at `modelcontextprotocol.io`
