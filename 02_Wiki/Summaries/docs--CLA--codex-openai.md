OpenAI Codex CLI 项目的个人贡献者许可协议（v1.0），基于 Apache 软件基金会 CLA v2.2。

**签署方式**：在 PR 中粘贴 "I have read the CLA Document and I hereby sign the CLA" 即视为同意，无需 Git 命令、邮件附件或 commit footer。

**版权许可**：贡献者授予 OpenAI 及项目接受者**永久、全球、非独占、免版税、不可撤销**的版权许可（复制、衍生、展示、分发、再许可等）。

**专利许可**：同等条款授予专利许可；若任一方发起针对项目或贡献内容的专利诉讼，该专利许可自动终止。

**贡献者声明**：贡献者须保证拥有合法授权提交代码；贡献内容以"原样"提供，不附带任何保证；若声明不准确须通知项目。

**管辖法律**：加利福尼亚州法律（不适用冲突法规则）。
