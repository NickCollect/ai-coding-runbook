OpenAI Codex CLI 是一个在本地计算机上运行的编码 agent，由 OpenAI 官方发布。

**安装方式**：支持 npm 全局安装（`npm install -g @openai/codex`）、Homebrew（`brew install --cask codex`）以及直接从 GitHub Releases 下载平台二进制包（macOS Apple Silicon/x86_64、Linux x86_64/arm64）。

**产品形态区分**：Codex CLI 是本地命令行工具；`codex app` 或 [chatgpt.com/codex](https://chatgpt.com/codex?app-landing-page=true) 提供桌面 App 体验；VS Code / Cursor / Windsurf IDE 插件是另一形态；Codex Web 则是 OpenAI 的云端 agent（非本 repo）。

**认证方式**：推荐通过 ChatGPT 账号登录（支持 Plus / Pro / Business / Edu / Enterprise 套餐），也可使用 API Key 但需额外配置。

**文档指引**：官方完整文档在 [developers.openai.com/codex](https://developers.openai.com/codex)；仓库内文档包括 contributing、install、open-source-fund 等。

**许可证**：Apache-2.0。
