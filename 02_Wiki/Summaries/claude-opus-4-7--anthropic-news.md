Claude Opus 4.7 generally available Apr 16, 2026. Notable upgrade over Opus 4.6 in advanced software engineering — gains concentrated on the hardest tasks. Users report being able to hand off their hardest coding work without close supervision; the model handles long-running tasks with rigor, follows instructions precisely, and devises ways to verify its own outputs.

**Other improvements.** Substantially better vision (higher image resolution). More tasteful and creative for professional outputs (interfaces, slides, docs). Better than Opus 4.6 across benchmarks but less broadly capable than Anthropic's most powerful model **Claude Mythos Preview**.

**Cyber safeguards** (first model in the post-Project-Glasswing rollout). During training, Anthropic experimented with differentially reducing cyber capabilities. Released with safeguards that automatically detect and block prohibited or high-risk cybersecurity uses. Security professionals can join the new **Cyber Verification Program** for legitimate use (vulnerability research, pentesting, red-teaming).

**Availability.** All Claude products + API + Amazon Bedrock + Google Cloud Vertex AI + Microsoft Foundry. Model ID `claude-opus-4-7`. Pricing unchanged from 4.6: $5 per M input tokens / $25 per M output tokens.

**Early-access partner highlights.** Hex: "low-effort Opus 4.7 ≈ medium-effort Opus 4.6." Cursor: cleared 70% on CursorBench vs. Opus 4.6's 58%. Notion: +14% over Opus 4.6 at fewer tokens and 1/3 the tool errors; first model to pass implicit-need tests; keeps executing through tool failures. Devin (Cognition): coherent autonomous work for hours. Replit: same quality at lower cost; pushes back during technical discussions. Solve Intelligence: better multimodal (chemical structures, technical diagrams). Harvey: 90.9% on BigLaw Bench at high effort.
