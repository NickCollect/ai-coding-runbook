`GET /v1/user_profiles/{user_profile_id}` — fetch a single user profile. **Path param:** `user_profile_id`. Requires `anthropic-beta` header (e.g. `managed-agents-2026-04-01` for the Managed Agents stack). Standard Anthropic API auth (`X-Api-Key`, `anthropic-version: 2023-06-01`) applies.
