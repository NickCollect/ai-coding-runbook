Joint announcement by David Soria Parra (MCP Lead Maintainer), Christopher Hertel (Symfony), and Roman Pronskiy (PHP Foundation) declaring **the official PHP SDK** generally available, built in collaboration with the PHP Foundation and Symfony.

**Initial scope**: PHP developers can build MCP **servers** exposing tools, prompts, and resources. **Client** support (PHP apps acting as MCP clients) will follow.

**Distribution**: source at `modelcontextprotocol/php-sdk`; Composer package at `mcp/sdk` on Packagist.

**Ecosystem**: PHP joins **9 other officially supported language SDKs** (Java, Ruby, Swift, Go, C#, Kotlin, Python, TypeScript, Rust pre-existing).

**Origins**: SDK consolidates earlier community work (Kyrian Obikwelu's PHP-MCP) into a single trusted implementation maintained by the Symfony team, with Kyrian joining as maintainer. PHP Foundation coordinated the initiative with the MCP steering group.

Welcomes community contributions especially framework-specific integrations and real-world examples.
