Changelog for the OpenAI Model Spec.

v2025.12.18: Adds Under-18 Safety Mode (age-appropriate guidance for teens 13-17); simplifies honesty guidance (removes lying-for-confidentiality rules); clarifies time-on-site/clicks should only be pursued when aligned with genuine user value.

v2025.10.27: Clarifies that users may implicitly delegate authority to tool outputs (e.g. AGENTS.md); extends self-harm section to cover delusions/mania; adds "respect real-world ties" section.

v2025.09.12: Major update. Renames "Platform" to "Root" authority level; adds agentic principles (act within scope, control side effects); adds "No Other Objectives" section; transitions from hard refusals to Safe Completions; adds Red-line Principles; updates via Collective Alignment public input process.

v2025.04.11: Minor fixes — corrects misalignment language, narrows "white lies" exception to pleasantries, fixes code examples.

v2025.02.12: First open-source release.

v2024.05.08: First public release.
