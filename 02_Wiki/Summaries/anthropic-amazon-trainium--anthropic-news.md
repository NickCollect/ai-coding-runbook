Nov 22, 2024 — **Amazon-Anthropic expansion: $4B additional investment** ($8B total). AWS confirmed as Anthropic's primary cloud and training partner.

**Trainium collaboration deepens.** Anthropic working with AWS Annapurna Labs on Trainium accelerator development/optimization. Writing low-level kernels directly interfacing Trainium silicon, contributing to AWS Neuron software stack. Hardware-software co-design from silicon up.

**Claude as core infrastructure.** Customer examples: Pfizer (medicine R&D + ops cost savings tens of millions), Intuit (tax-calculation explanation for millions during tax season), Perplexity (2× faster more accurate responses), European Parliament Archibot (2.1M docs searchable in multiple languages, 80% research-time reduction).

**Government access.** AWS GovCloud (US) for federal customers; AWS Secret/Top Secret Cloud Regions for highly controlled environments (via Amazon SageMaker).
