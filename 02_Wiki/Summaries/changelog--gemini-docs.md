# Gemini API Changelog

Source is in Vietnamese (crawler localization). Covers release notes up to May 2026.

## Key Recent Releases (Reverse Chronological)

### May 4, 2026
- Webhooks support launched for Gemini API — event-based alternative to polling for Batch API and long-running operations.

### April 30, 2026
- `gemini-robotics-er-1.5-preview` shut down → use `gemini-robotics-er-1.6-preview`.

### April 22, 2026
- `gemini-embedding-2` GA released.

### April 21, 2026
- New Deep Research agents: `deep-research-preview-04-2026` (fast, for streaming to UI), `deep-research-max-preview-04-2026` (maximum coverage). Features: collaborative planning, visualization, MCP server integration, File Search.

### April 15, 2026
- Gemini 3.1 Flash TTS Preview launched (`gemini-3.1-flash-tts-preview`).

### April 14, 2026
- `gemini-robotics-er-1.6-preview` released with instrument reading, improved spatial/physical reasoning.

### April 2, 2026
- Gemma 4 launch: `gemma-4-26b-a4b-it` and `gemma-4-31b-it` available on AI Studio and via Gemini API.

### April 1, 2026
- New inference tiers: Flex (cost-optimized, variable latency) and Priority (low-latency SLA).

### March 31, 2026
- Veo 3.1 Lite preview (`veo-3.1-lite-generate-preview`) launched.
- `gemini-2.5-flash-lite-preview-09-2025` shut down → use `gemini-3.1-flash-lite-preview`.

### March 26, 2026
- `gemini-3.1-flash-live-preview` released — new A2A model for real-time voice/dialogue.

### March 25, 2026
- Lyria 3 music generation models launched: `lyria-3-clip-preview` (30s clips), `lyria-3-pro-preview` (full songs). Accept text + image input, output 48kHz stereo audio.

### March 23, 2026
- Prepay and postpay billing plans launched in AI Studio.

### March 18, 2026
- Tool combination feature: built-in Gemini tools + custom function calling in a single API call.
- Maps Grounding now supported for Gemini 3 models.

### March 16, 2026
- Improved billing tiers and billing account spend caps.

## Notes
- Full changelog continues back through earlier 2026 and 2025 releases.
- Deprecation announcements appear in changelog first, then tracked on the Deprecations page.
