TypeScript type reference for the Realtime API resource (client.realtime). Contains extensive types for all WebSocket events: ConversationItem, audio buffer events (InputAudioBufferAppendEvent, InputAudioBufferCommitEvent, InputAudioBufferSpeechStartedEvent, etc.), transcription events, and session/response types. See realtime.md for usage guide.
