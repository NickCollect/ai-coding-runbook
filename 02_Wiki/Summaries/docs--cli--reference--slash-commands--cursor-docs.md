Cursor CLI 交互模式中可用的所有 Slash Commands 速查表。

**模式切换**：`/plan`（Plan 模式）、`/ask`（Ask 只读模式）。

**模型**：`/model <model>`（设置或列出模型）、`/max-mode [on|off]`（Max Mode 开关）。

**会话管理**：`/new-chat`（新会话）、`/resume <chat>`（恢复历史会话）、`/compress`（压缩对话释放上下文空间）。

**工具配置**：`/auto-run [on|off|status]`（自动执行开关）、`/sandbox`（沙盒模式和网络设置）、`/mcp list/enable/disable <name>`（MCP 服务器管理）。

**代码工具**：`/rules`（创建或编辑 Rule）、`/commands`（创建或编辑 Commands）。

**界面**：`/vim`（Vim 键位开关）、`/setup-terminal`（终端快捷键自动配置）。

**信息**：`/usage`（查看使用统计和连续使用天数）、`/about`（环境和 CLI 设置详情）、`/copy-request-id`、`/copy-conversation-id`、`/feedback <message>`、`/help [command]`、`/logout`、`/quit`。
