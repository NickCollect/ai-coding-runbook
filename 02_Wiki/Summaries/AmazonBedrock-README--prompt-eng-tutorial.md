Bedrock Edition of the standalone Prompt Engineering Interactive Tutorial repo. Same content as the version in the `courses/` repo's Bedrock subdirectory.

**Outcomes.** Same as the 1P version: master basic prompt structure, recognize common failure modes and 80/20 fixes, understand Claude's strengths/weaknesses, build strong prompts from scratch.

**Structure.** 9 chapters with exercises plus an appendix. Each lesson has an "Example Playground" area. The 1P answer key applies — solutions are the same.

Note: defaults to Claude 3 Haiku.

**Table of contents:**

- **Beginner** — Chapter 1: Basic Prompt Structure; Chapter 2: Being Clear and Direct; Chapter 3: Assigning Roles.
- **Intermediate** — Chapter 4: Separating Data from Instructions; Chapter 5: Formatting Output & Speaking for Claude; Chapter 6: Precognition (Thinking Step by Step); Chapter 7: Using Examples.
- **Advanced** — Chapter 8: Avoiding Hallucinations; Chapter 9: Building Complex Prompts (Industry Use Cases) including Chatbot, Legal Services, Financial Services exercise, Coding exercise, Congratulations & Next Steps.
- **Appendix: Beyond Standard Prompting** — Chaining Prompts, Tool Use, Empirical Performance Evaluations, Search & Retrieval. (Adds "Empirical Performance Evaluations" relative to the 1P appendix.)

The substantive content is identical to the 1P edition; only the API target (Bedrock vs. direct Claude API) changes.
