What's-new digest for April 13–17, 2026 (releases v2.1.105 → v2.1.113). Five headline features.

**Claude Opus 4.7** — Anthropic's strongest coding model. Default on Max + Team Premium. Adds new `xhigh` effort level (between `high` and `max`) — applied as default first time you switch to 4.7. `/effort` with no args opens interactive arrow-key slider.
```
> /model opus
> /effort xhigh
```

**Routines** (web) — templated cloud agents on schedule / GitHub event / API call. Define on Claude Code on the web with prompt, repos, connectors. Trigger picker covers GitHub events with optional filters; every routine gets tokened `/fire` endpoint for external systems. CLI: `/schedule daily PR review at 9am`.

**`/usage` breakdown** — shows what's driving your limits: parallel sessions, subagents, cache misses, long context, each with % of last 24 hours and an optimization tip. Press `d`/`w` to toggle day/week views.

**`/ultrareview`** (v2.1.111) — comprehensive cloud code review. Fans branch out across parallel reviewers on Claude Code on the web, runs adversarial critique pass, returns verified findings while terminal stays free. Call no-args (current branch) or `/ultrareview 1234` (PR). Launch dialog shows diffstat.

**Native binaries** (v2.1.113) — `claude` CLI now spawns native per-platform binary instead of bundled JavaScript. npm package pulls right binary via optional dep (e.g., `@anthropic-ai/claude-code-darwin-arm64`); install command unchanged. Standalone installer already shipped this; npm now matches.

**Other wins**:
- Auto mode now available for Max subscribers on Opus 4.7; `--enable-auto-mode` flag no longer required
- **Session recap** — one-line summary of what happened while away. Run `/recap` on demand or toggle from `/config`
- New `/tui` command and `tui` setting switch between classic and flicker-free rendering mid-conversation; **focus view moved from `Ctrl+O` to its own `/focus` command**
- **Push notification tool**: with Remote Control connected and "Push when Claude decides" enabled, Claude can ping your phone when it needs you
- Plugins can ship background watchers via top-level `monitors` manifest key (auto-arm at session start or on skill invoke)
- "Auto (match terminal)" option in `/theme` follows terminal's dark/light mode
- `/fewer-permission-prompts` scans transcripts for common read-only Bash + MCP calls, proposes allowlist for `.claude/settings.json`
- Claude can discover/run built-in commands like `/init`, `/review`, `/security-review` via the Skill tool
- `PreCompact` hooks can BLOCK compaction (exit code 2 or `{"decision":"block"}`)
- `ENABLE_PROMPT_CACHING_1H` opts API key/Bedrock/Vertex/Foundry users into 1-hour prompt cache TTL
- `sandbox.network.deniedDomains` carves specific domains out of broader `allowedDomains` wildcard
- `/undo` alias for `/rewind`, `/proactive` alias for `/loop`
- Hardened Bash permissions: deny rules now match through `env`/`sudo`/`watch` wrappers; `Bash(find:*)` allow rules no longer auto-approve `-exec`/`-delete`
