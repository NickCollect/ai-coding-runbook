Claude Opus 4.7 是 Anthropic 最强模型，在 CursorBench 上显著超越 Opus 4.6，擅长自主多步骤任务、创意推理和复杂规划，推荐使用 high thinking 变体。

**优势**：自主驱动多步骤任务至完成，跨大代码库和长对话不丢失目标；创意推理探索非预期解法；规划先行，提前捕获边界情况；工具调用有针对性并链式跟进。

**局限**：最贵的模型，用量消耗快；长会话中可能过度阐述。

**计费**：API 用量池；所有 Opus 4.7 prompt（包括 Max Mode 超 200k token 时）均按同一基础费率计费，无长上下文乘数，最高支持 1M token。支持 Thinking 变体，推荐 high thinking 设置。
