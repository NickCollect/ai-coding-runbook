GitHub issue template for the anthropics/claude-code repo. Structured form for community feature requests. Required fields: preflight checklist (search existing requests, single feature only), Problem Statement, Proposed Solution, Priority dropdown (Critical / High / Medium / Low), Feature Category dropdown (CLI commands and flags / Interactive mode TUI / File ops / API and model interactions / MCP server integration / Performance / Configuration / Developer tools/SDK / Documentation / Other). Optional: Alternative Solutions, Use Case Example, Additional Context.

Auto-prefixes title with `[FEATURE]`, applies `enhancement` label.
