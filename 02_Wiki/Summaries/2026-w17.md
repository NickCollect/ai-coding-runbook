Weekly digest covering Claude Code v2.1.114 → v2.1.119 (April 20–24, 2026). Four featured items + many wins.

**Featured**:

- **`/ultrareview` (public research preview)**: cloud-based multi-agent bug-hunting reviewer; findings stream back to CLI/Desktop. Run before merging critical changes (auth, data migrations).
  ```text
  /ultrareview        # current branch vs default
  /ultrareview 1234   # PR #1234
  ```

- **Session recap (CLI)**: switch focus away → return → one-line recap of what happened. Useful when running multiple sessions. `/recap` to generate on demand; auto recap toggleable in `/config`.

- **Custom themes (v2.1.118)**: build/switch named color themes from `/theme`, or hand-edit JSON in `~/.claude/themes/`. Each theme picks a base preset and overrides only the tokens you care about. Plugins can ship themes via `themes/` directory.

- **Claude Code on the web redesigned**: new look matching desktop app — sessions sidebar, drag-and-drop layout, refreshed routines view. Performance + reliability rebuild.

**Other wins**:
- **Vim visual mode**: `v` (character selection) / `V` (line selection) in prompt input, with operators
- **Hooks can call MCP tools directly** via `type: "mcp_tool"` — no process spawn, hits already-connected server
- **`/cost` + `/stats` merged into `/usage`**; old names remain typing shortcuts opening relevant tab
- **`/config` changes persist to `~/.claude/settings.json`** and follow project/local/policy precedence (theme, editor mode, verbose, etc.)
- **Forked subagents** on external builds via `CLAUDE_CODE_FORK_SUBAGENT=1` — fork inherits full conversation context instead of starting fresh
- **Default effort level** raised to `high` for Pro/Max subscribers on Opus 4.6 + Sonnet 4.6 (was `medium`)
- **Native macOS/Linux builds**: Glob and Grep replaced by embedded `bfs` and `ugrep` accessible through Bash → faster searches, no separate tool round-trip
- **`--from-pr`** accepts GitLab MR / Bitbucket PR / GitHub Enterprise PR URLs in addition to github.com
- **Auto mode `"$defaults"` literal** in `autoMode.allow`/`soft_deny`/`environment` arrays inherits built-in rules instead of replacing them
- **`claude plugin tag`** new command creates release git tags for plugins with version validation
- **Opus 4.7 sessions** now compute against the model's native 1M context window, fixing inflated `/context` percentages and premature autocompaction
- **`/resume` on large sessions up to 67% faster**; offers to summarize stale large sessions before re-reading
