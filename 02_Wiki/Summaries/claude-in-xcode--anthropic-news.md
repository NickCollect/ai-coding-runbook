Sep 15, 2025 — **Claude Sonnet 4 generally available in Xcode 26**, Apple's IDE for iOS/Mac/iPadOS/watchOS/visionOS app development. Available to all plans that include Claude Code (Pro, Max, Team Premium seats, Enterprise Premium seats). Subscription usage limits shared across platforms.

**Features.**
- *Coding assistant* — natural-language interaction; auto-gathers project context; supports conversation history and file attachments for debugging, refactoring, building features.
- *Coding tools* — generate documentation, explain highlighted code sections, create SwiftUI previews and playgrounds, inline edits.

Setup: download Xcode 26 from Mac App Store → Xcode preferences → Intelligence settings → log in with Claude account. Basis for the later Xcode 26.3 deeper integration with the Claude Agent SDK.
