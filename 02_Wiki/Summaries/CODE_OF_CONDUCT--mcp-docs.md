Standard **Contributor Covenant 2.0** Code of Conduct used by the MCP docs repo. Pledges harassment-free participation regardless of age, body size, disability, ethnicity, gender, etc. Lists positive behaviors (empathy, respect, constructive feedback, accountability, community-first focus) and unacceptable behaviors (sexualized language, trolling, harassment, doxxing, unprofessional conduct).

Community leaders enforce standards and may remove/edit/reject contributions misaligned with the CoC. Scope covers all community spaces and any context where someone officially represents the community.

**Enforcement contact**: `mcp-coc@anthropic.com`. The four-step enforcement ladder is the standard Contributor Covenant: (1) Correction (private written warning), (2) Warning (consequences for continued behavior + no-contact period), (3) Temporary Ban, (4) Permanent Ban for repeated patterns or harassment.

Adapted from Contributor Covenant 2.0 with Mozilla-inspired enforcement guidelines.
