Changelog of `@anthropic-ai/sdk` for TypeScript. Recent significant releases:

**0.93.0 (2026-05-04).** Client: added Workload Identity Federation, interactive OAuth, and auth profiles.

**0.92.0 (2026-04-30).** API: improved Managed Agents APIs. Client: support setting headers via env. Bedrock fix: throw `APIError` for error events delivered in chunk frames.

**0.91.1 (2026-04-24).** Memory bug fix: use restrictive file mode for memory files.

**0.91.0 (2026-04-23).** API: CMA Memory public beta. Bedrock: switched mantle client to use auth header. API spec error fixes and restored missing features.

**0.90.0 (2026-04-16).** Added `claude-opus-4-7`, token budgets, and `user_profiles` to the API surface.

**0.89.0 (2026-04-14).** API manual updates; marked Sonnet and Opus 4 as deprecated. Streaming bug fix — added missing events.

**0.88.0 (2026-04-10).** Vertex EU region support. Documentation example improvements.

**0.87.0 (2026-04-09).** Added beta advisor tool support.
