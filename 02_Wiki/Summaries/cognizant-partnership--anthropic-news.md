Nov 4, 2025 — **Cognizant deploys Claude to up to 350,000 global employees**. Combines Claude with Cognizant's engineering platforms and industry blueprints. Internal use of Claude Code for coding, testing, documentation, DevOps. Aligns Claude models, Claude Code, MCP, Agent SDK with Cognizant's core platforms so clients can integrate AI with existing data/apps, orchestrate multi-step work with human oversight, manage performance/risk/spend.
