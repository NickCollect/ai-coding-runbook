Cursor Enterprise 为大规模部署 AI 辅助开发的组织提供企业级安全、合规和管理控制，持有 SOC2 Type II 认证和 GDPR 合规。

**核心功能域**：身份与访问（SSO/SAML、SCIM、RBAC、MDM 策略）、隐私与数据治理（Privacy Mode 零数据保留、数据驻留）、网络配置（代理/IP 白名单/加密）、LLM 安全控制（Hooks/终端沙盒/Agent 控制）、模型与集成管理（模型访问限制/MCP 管理）、合规与监控（审计日志/SIEM 集成）。

**Enterprise 专属功能**：
- 用量池共享（pooled usage）
- Billing Groups（部门费用分摊）
- Service Accounts（非人类账号自动化）
- SCIM 用户供应
- Audit Log（防篡改安全事件记录）
- AI Code Tracking API、Conversation Insights
- Cursor Blame（AI vs 人工代码归因）
- 仓库 Blocklist、模型访问控制
- Auto-run/浏览器/网络企业级管控
- 用户级 CLI/Cloud Agent/Analytics 访问控制
- HIPAA BAA、优先技术支持（严重事故 8 小时、标准 24 小时）

**计划对比**：Individual（个人限制）→ Teams（SSO/团队分析/隐私模式强制/1 个团队 Marketplace）→ Enterprise（上述全部 + 无限 Marketplace/SCIM/审计/高级控制）。
