GPT-5.5 是 Cursor 中 OpenAI 最高智能模型，擅长长时运行任务，比同类模型更少提前停止，适合复杂多步骤编程会话。

**优势**：GPT-5 系列最高智能；长时任务持续性强，任务未完成时继续执行而非提早结束；跨大型多步骤工作流维持上下文质量；复杂任务 Token 效率高于 GPT-5.4。

**局限**：每 token 价格高于 GPT-5.3 Codex；简单任务吞吐量相对低。

**计费**：API 用量池；超过 272k token（长上下文/Max Mode）时 input 定价翻倍，output 为 1.5x；提供 `gpt-5.5-fast` 快速模式（更高 token 价）。
