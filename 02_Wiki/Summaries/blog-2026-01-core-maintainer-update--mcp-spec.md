David Soria Parra announces Core Maintainer team changes for 2026.

**Departing**: Inna Harper and Basil Hosmer step away from Core Maintainer role to focus on other projects. Both involved since the early days — key design decisions, SDK development, all major spec releases.

**New Core Maintainers** (3):

- **Peter Alexander** — Member of Technical Staff at Anthropic on the MCP team. Prior: Senior Staff at Meta (VR, video conferencing, real-time pub/sub data infrastructure), gameplay lead at Codemasters (Grid, DiRT). Authored the Extensions framework SEP and Resource Contents Metadata SEP; active reviewer of SDKs/governance/spec proposals.
- **Caitie McCaffrey** — Software Engineer / Tech Lead at Microsoft. Distributed systems experience across AI platforms, gaming, social media, IoT. Previously Technical Advisor to Satya Nadella and Kevin Scott; observability lead at Twitter; gaming work on Halo 4/5 and Gears of War 2/3. Reviewing governance and transport proposals.
- **Kurtis Van Gent** — Senior Staff Engineer leading AI Ecosystems at Google Cloud Databases. Drives MCP for Data Cloud and MCP Toolbox for Databases. Driving Transport Working Group; authored the request payload decoupling SEP and leads the stateless protocol proposal.

**Looking ahead 2026**: active SEPs include DPoP extension for auth, multi-turn SSE for transport, Server Cards for discovery. Focus areas: spec hardening for enterprise, security/auth patterns, SDK implementation guidance, scale.
