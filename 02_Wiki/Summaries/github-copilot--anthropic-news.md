Oct 29, 2024 — **Claude 3.5 Sonnet rolls out on GitHub Copilot** in public preview. Available in VS Code and GitHub.com to GitHub's 100M+ developer community via GitHub Copilot Chat. Runs through Amazon Bedrock cross-region inference.

**Performance basis.** Upgraded Claude 3.5 Sonnet leads SWE-bench Verified; 93.7% on HumanEval (top in class).

**Use cases.** Production-ready code generation matching project conventions, debugging via inline chat with code-block highlighting, automatic test-suite generation from selected functions, contextual code explanations on hover.

The rollout established Claude as one of GitHub Copilot's primary model choices (alongside OpenAI models), giving Anthropic mass distribution to developer audiences.
