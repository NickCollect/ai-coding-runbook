Claude Sonnet 4.6 是 Anthropic 中等智力层级模型，与 Sonnet 4.5 价格相同，支持 Thinking 模式，默认 200k 上下文，Max Mode 可扩展至 1M。

**优势**：比 Opus 价格低但保留强编程能力和 Thinking 支持；适合日常编码任务；同 Anthropic 提供商。

**局限**：性价比不如 Codex 和 Composer；巅峰质量仍以 Opus 为优。

**计费**：从 API 用量池消耗（非 Auto 池）；超过 200k token（长上下文）时定价翻倍；支持 Thinking 变体。
