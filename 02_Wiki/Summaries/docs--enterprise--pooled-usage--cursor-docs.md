Pooled Usage 为 Enterprise 团队提供共享用量池替代固定人均分配，全团队从合同期内预承诺的统一预算中扣减，资源按需分配。

**核心价值**：无固定人均限制 → 不会出现某人用量耗尽而其他人额度闲置的情况；需求最多的项目可使用全团队预算。

**工作方式**：团队对合同期总用量做预承诺；所有成员共享此池；用量在整个合同期内累计追踪（非按月重置）。

**管理功能**：
- **Dynamic Spend Limits**：随团队规模自动扩缩的全团队花费上限
- **Spend Alerts**：设置阈值，达到时发送邮件通知（不阻断工作）
- **Admin Controls**：仅管理员可调整用量池设置

Enterprise 专属功能，需联系销售配置。
