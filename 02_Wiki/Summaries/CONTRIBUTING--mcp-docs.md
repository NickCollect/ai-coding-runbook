Contributor guide for the (deprecated) `modelcontextprotocol/docs` repo.

**Setup prerequisites**: Node.js, Git, Mintlify CLI (`npm i -g mintlify`). Fork → clone → branch → run `mintlify dev` to preview locally.

**Documentation guidelines**: clear/concise/accurate content; follow existing structure and naming; include code examples; use proper MDX components; test links and code samples.

**Content types** accepted: conceptual guides, API reference docs, tutorials and quickstarts, best practices and examples, troubleshooting guides.

**Pull request process**: follow guidelines, update relevant ToCs, test with `mintlify dev`, submit PR with clear description, address review feedback.

**Resources**: existing docs at `modelcontextprotocol.io`, issue tracker at `github.com/modelcontextprotocol/docs/issues`, community discussions on the same repo. Project follows the standard Contributor Covenant Code of Conduct (linked).
