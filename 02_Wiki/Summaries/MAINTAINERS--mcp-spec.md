Roster of current MCP maintainers (last updated April 8, 2026). Documents the org chart of the project.

**Lead Maintainers**: David Soria Parra (`@dsp-ant`), Den Delimarsky (`@localden`).

**Core Maintainers** (8): Caitie McCaffrey, Che Liu, Clare Liguori, Kurtis Van Gent, Nick Aldridge, Nick Cooper, Paul Carleton, Peter Alexander.

**Emeritus**: Justin Spahr-Summers (Co-Inventor, Lead Maintainer Emeritus).

**SDK Maintainers** (per language): Java, Ruby, Swift, Go, C#, Kotlin, Python (Inna Harper, Jerome Swannack, Marcelo Trylesinski, Max Isbey, Felix Weinberger), TypeScript (Inna Harper, Felix Weinberger, Olivier Chafik, Konstantin Konstantinov, Matt Carey), Rust, PHP — each SDK has 4–7 maintainers.

**Project Maintainers** for ecosystem subprojects: `use-mcp` (Glen Maddern); **Inspector**; **Registry** (Toby Padilla, Tadas Antanavicius, Adam Jones, Radoslav Dimitrov, Bob Dickinson); **MCPB** / Model Context Protocol Bundle (Alexander Sklar, Adam Jones, Joan Xie); **Reference Servers**.

**Community Moderators**: Ola Hungerford, Cliff Hall, Shaun Smith, Jonathan Hefner, Tadas Antanavicius, Peder Holdgaard Pedersen.

**Working Group / Interest Group Maintainers**:
- Security Interest Group, Authorization Interest Group
- **Client Implementor Interest Group** — protocol representatives for Goose, Zed, VS Code, Codex, GitHub Copilot for JetBrains, Cursor (note: client-specific issues should go through each product's official support channels, not these contacts)
- Financial Services, Transports
- Server Identity Working Group, Agents Working Group, MCP Apps Working Group
- Primitive Grouping Interest Group

Working/Interest Groups are not required to have formal maintainers (informal facilitators allowed); maintainers may be appointed as needed.
