`GET /v1/memory_stores` — paginated list of memory stores. Cursor pagination consistent with peer beta resources. Requires `anthropic-beta` header (e.g. `managed-agents-2026-04-01` for the Managed Agents stack). Standard Anthropic API auth (`X-Api-Key`, `anthropic-version: 2023-06-01`) applies.
