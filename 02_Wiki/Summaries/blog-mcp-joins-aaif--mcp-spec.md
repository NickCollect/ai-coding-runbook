Anthropic donates **Model Context Protocol** to the **Agentic AI Foundation (AAIF)** — a directed fund under the **Linux Foundation** — making MCP a founding project of AAIF. David Soria Parra writes the announcement.

**Year-one stats** quoted: 97M+ monthly SDK downloads, 10,000+ active servers, first-class client support across ChatGPT, Claude, Cursor, Gemini, Microsoft Copilot, Visual Studio Code, etc.

**Why the move**: formalizes the long-standing commitment to keep MCP open and community-driven. Vendor neutrality now backed by the same Linux Foundation stewardship that supports Kubernetes, PyTorch, Node.js. Anthropic's commitment is unchanged: continued investment, infrastructure maintenance, active community participation.

**About AAIF**: directed fund under Linux Foundation, co-founded by Anthropic, Block, OpenAI, with support from Google, Microsoft, AWS, Cloudflare, Bloomberg. MCP joins **goose** (by Block) and **AGENTS.md** (by OpenAI) as founding projects. AAIF Governing Board handles strategic investments, budget, member recruitment, project approval — but **individual projects retain full autonomy over technical direction and day-to-day operations**.

**MCP governance unchanged**: same maintainer structure, same SEP process, same governance model from earlier in 2025. Linux Foundation provides neutral home and infrastructure; doesn't dictate technical direction.
