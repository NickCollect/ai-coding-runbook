Dec 3, 2025 — **$200M multi-year Snowflake-Anthropic partnership**. Claude available to Snowflake's 12,600+ global customers across Bedrock, Vertex AI, Azure. Joint go-to-market for AI agents in world's largest enterprises.

**What it delivers.**
- **Claude Sonnet 4.5 powers Snowflake Intelligence** — enterprise intelligence agent answering NL questions over structured + unstructured data.
- **Cortex AI Functions** — Claude (including Opus 4.5 hosted day-one) for multimodal analysis: text, images, audio, tabular via SQL.
- **Cortex Agents** — production-ready data agents powered by Claude.
- **Horizon Catalog** — governance and observability for moving regulated-industry agents from pilot to production.

Snowflake customers process trillions of Claude tokens/month via Cortex AI today. Snowflake also uses Claude Code internally; built a Claude-powered GTM AI Assistant on Snowflake Intelligence. **>90% accuracy on complex text-to-SQL** in Snowflake's internal benchmarks. Customer example: Simon Data uses Claude on Snowflake for hidden-pattern discovery.
