@anthropic-ai/bedrock-sdk: Claude API via AWS Bedrock. Install: npm install @anthropic-ai/bedrock-sdk. Client class: AnthropicBedrock. Uses AWS credential chain (~/.aws/credentials or AWS_ACCESS_KEY_ID env vars). Model IDs use Bedrock format (e.g. anthropic.claude-3-5-sonnet-20241022-v2:0). See docs at platform.claude.com/docs/en/build-with-claude/claude-on-amazon-bedrock.
