Important: **this repository is NOT eligible for security vulnerability reporting**. The servers here are reference implementations meant for education, not production. Vulnerabilities specific to these example servers should NOT be reported as security issues.

If you discover a vulnerability in an **MCP SDK**, report it through the GitHub Security Advisory process in the relevant SDK repository. Do NOT use public GitHub issues / discussions / PRs for security issues.

For the broader MCP project trust model and intended-behavior list, see `SECURITY--mcp-spec`.
