Composer 2 是 Cursor 自研 Agent 模型，在 CursorBench 上达到前沿编程水准，Token 效率高，专为 Cursor 内工具调用、文件编辑和终端操作优化。

**优势**：训练时采用自摘要（self-summarization）技术，擅长长时任务；针对工具调用、文件编辑、终端操作深度优化；交互会话用更快变体为默认。

**计费**：个人计划使用独立 Composer 用量池（包含慷慨额度）；团队/企业计划按 API 价格计费。提供更快变体（$1.50/M input，$7.50/M output），标准变体价格更低。Auto 模式也可能使用 Composer。
