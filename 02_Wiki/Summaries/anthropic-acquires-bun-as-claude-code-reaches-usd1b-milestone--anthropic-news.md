Dec 3, 2025 — **Anthropic acquires Bun**, a JavaScript runtime / all-in-one toolkit (runtime + package manager + bundler + test runner) founded by Jarred Sumner in 2021. Acquisition closes in the same announcement that **Claude Code reached $1B run-rate revenue** in 6 months from public availability (May 2025).

**Why Bun.** Bun has been instrumental in scaling Claude Code infrastructure and directly drove the recent launch of Claude Code's native installer. CPO Mike Krieger: "Jarred and his team rethought the entire JavaScript toolchain from first principles." Bun has 7M+ monthly downloads, 82,000+ GitHub stars; adopted by Midjourney, Lovable.

**Customer footprint.** Claude Code now used by Netflix, Spotify, KPMG, L'Oreal, Salesforce. Bun will remain open source and MIT-licensed; Anthropic will keep investing in it as the runtime/bundler/package-manager/test-runner of choice.
