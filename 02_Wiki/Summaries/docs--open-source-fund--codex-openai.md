OpenAI 面向使用 Codex CLI 及其他 OpenAI 模型的开源项目推出 **$100 万激励计划**。

**资助额度**：单项最高 **$25,000** API 积分。

**申请流程**：滚动评审（随时可申请），申请入口：[openai.com/form/codex-open-source-fund/](https://openai.com/form/codex-open-source-fund/)。
