Cursor 是 AI 编辑器和编程 Agent，可用于理解代码库、规划和构建功能、修复 Bug、审查变更，以及与现有工具集成。

**核心用途**：理解代码库（追踪项目结构）、规划和构建功能（Plan Mode）、发现和修复 Bug、审查变更（Diff 审查和检查）、自定义 Cursor（Rules/Skills/Prompts）、连接工作流（GitHub/GitLab/JetBrains/Slack/Linear 集成）。

**支持的模型**（含上下文窗口）：Claude 系列（4 Sonnet/4.5/4.6/4.7 Opus/Sonnet，默认 200k，Max Mode 最大 1M）；Composer 2（200k，Cursor 自研）；Gemini 3 系列（200k，Max Mode 1M）；GPT-5 系列（272k）；Grok 4.20（200k，Max Mode 2M）；Kimi K2.5（262k）。

文档首页作为导航枢纽，链接到 Quickstart、Models & Pricing、Changelog 以及各功能文档。
