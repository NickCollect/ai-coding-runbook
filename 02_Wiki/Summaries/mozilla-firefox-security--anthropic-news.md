Mar 6, 2026 — **Anthropic-Mozilla collaboration**: over 2 weeks, Claude Opus 4.6 found **22 vulnerabilities in Firefox**, of which Mozilla classified **14 as high-severity** — almost a fifth of all high-severity Firefox vulnerabilities remediated in 2025. Fixes shipped to hundreds of millions of users in Firefox 148.0.

**Methodology.** Built a Firefox-CVE benchmark from prior CVEs to test if Claude could reproduce them. Firefox chosen because complex + well-tested + browser-vulnerabilities particularly dangerous. Mozilla helped Anthropic understand what findings warranted bug reports. Provides a model for AI-enabled security researchers + maintainers collaboration. Builds on prior 500-zero-days finding.
