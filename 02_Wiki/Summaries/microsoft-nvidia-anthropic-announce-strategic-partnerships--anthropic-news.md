Nov 18, 2025 — Three-way **Microsoft + NVIDIA + Anthropic** partnership announced.

**Headlines.**
- Anthropic to scale Claude on Azure (commits to **$30B of Azure compute**, contracted up to 1 GW capacity).
- Anthropic to adopt NVIDIA architecture — initially up to 1 GW of compute capacity on Grace Blackwell and Vera Rubin systems. Joint design/engineering optimizing Claude for NVIDIA performance/efficiency/TCO and future NVIDIA architectures for Anthropic workloads.
- **NVIDIA invests up to $10B in Anthropic; Microsoft invests up to $5B**.
- Microsoft Foundry gets Sonnet 4.5, Opus 4.1, Haiku 4.5. Microsoft 365 Copilot, GitHub Copilot, Copilot Studio continue Claude access.
- **Claude becomes the only frontier model on all three major clouds** (AWS Bedrock + Vertex AI + Azure Foundry).

Amazon remains Anthropic's primary cloud and training partner.
