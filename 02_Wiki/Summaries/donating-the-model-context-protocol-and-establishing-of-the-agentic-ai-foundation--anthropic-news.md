Dec 9, 2025 — **Anthropic donates MCP to the Linux Foundation's new Agentic AI Foundation (AAIF)**. Co-founded by Anthropic, Block, OpenAI; supported by Google, Microsoft, AWS, Cloudflare, Bloomberg.

**MCP one year on (since Nov 2024 launch).**
- 10,000+ active public MCP servers.
- Adopted by ChatGPT, Cursor, Gemini, Microsoft Copilot, VS Code.
- Enterprise infra support from AWS, Cloudflare, Google Cloud, Microsoft Azure.
- 75+ Claude connectors (claude.com/connectors).
- Tool Search and Programmatic Tool Calling (advanced-tool-use launch, Nov 24) optimize production-scale MCP deployments.
- Official community Registry for discovery (github.com/modelcontextprotocol/registry).
- Nov 25 spec release added async ops, statelessness, server identity, official extensions.
- 97M+ monthly SDK downloads (Python + TS).

**AAIF.** Directed fund under Linux Foundation aiming to ensure agentic AI evolves transparently, collaboratively, in public interest. Founding projects: Block's `goose`, OpenAI's `AGENTS.md`, and now Anthropic's MCP. MCP governance unchanged — community-driven, transparent decisions retained.
