TypeScript type reference for the Responses API (client.responses). Types include: Response, computer use tools (ComputerTool, ComputerAction, ComputerUsePreviewTool), FileSearchTool, FunctionTool, FunctionShellTool, container types (ContainerAuto, ContainerReference, ContainerNetworkPolicy*), response event types (ResponseAudioDeltaEvent, ResponseAudioDoneEvent, etc.), skills (InlineSkill, LocalSkill, NamespaceTool), ApplyPatchTool, CustomTool. Primary interface for text generation.
