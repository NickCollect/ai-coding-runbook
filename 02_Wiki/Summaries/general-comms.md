Fallback guideline file inside `internal-comms/examples/`. Used when the requested internal communication doesn't fit the standard formats (3P updates, newsletters, FAQs).

**Pre-write interview**: ask user about target audience, communication's purpose, desired tone (formal / casual / urgent / informational), specific formatting requirements.

**General principles**:
- Be clear and concise
- Use active voice
- Put most important information first
- Include relevant links and references
- Match the company's communication style
