Aug 20, 2025 — **Claude Code added to Team and Enterprise plans** as a premium-seat upgrade combining Claude app + Claude Code under one subscription. Admin gets visibility/controls; users move seamlessly between ideation (Claude) and implementation (Claude Code).

**Admin features.**
- *Self-serve seat management* — purchase, allocate, provision via admin panel.
- *Granular spend controls* — org and per-user spending limits.
- *Usage analytics* — Claude Code metrics (lines accepted, suggestion accept rate, usage patterns).
- *Managed policy settings* — deploy/enforce settings across users: tool permissions, file-access restrictions, MCP server configurations.

**Compliance API.** Real-time programmatic access to usage data and customer content for continuous monitoring, automated policy enforcement, integration with existing compliance dashboards, selective deletion for data-retention.

**Pricing model.** Standard usage included; admins can enable extra usage at standard API rates with per-user spend caps.

Customer wins: Behavox (compliance/security): bundled with Claude for Enterprise, "go-to pair programmer" for hundreds of developers. Altana (supply-chain AI Product Network): 2-10× development velocity, taking on more ambitious projects.
