Claude Opus 4.5 launched Nov 24, 2025. Best in the world for coding, agents, and computer-use; meaningfully better at deep research, slides, spreadsheets. **Pricing dropped to $5/$25 per M tokens** (from prior Opus pricing of $15/$75) — making Opus-level capabilities accessible to far more users.

**Released alongside Opus 4.5:**
- Claude Developer Platform updates.
- Claude Code updates.
- New consumer-app capabilities.
- Tools for longer-running agents.
- New Excel, Chrome, desktop integrations.
- Lengthy conversations no longer hit walls in the apps.

**Availability.** Apps + API + all three major cloud platforms. Model ID `claude-opus-4-5-20251101`.

**State-of-the-art on real-world software engineering.** Testers report Opus 4.5 "gets it" — handles ambiguity, reasons about tradeoffs without hand-holding, fixes complex multi-system bugs.

**Partner highlights.**
- GitHub Copilot: "surpasses internal coding benchmarks while cutting token usage in half"; well-suited for code migration/refactoring.
- Cursor: "notable improvement... improved pricing and intelligence on difficult coding tasks."
- Lovable: "frontier reasoning within Lovable's chat mode."
- Warp: 15% improvement over Sonnet 4.5 on Terminal-Bench.
- Manus: "self-improving AI agents" — agents reach peak performance in 4 iterations vs. competitors' 10.
- Augment: 65% fewer tokens at higher pass rates.
- Cognition (Devin): refactor across 2 codebases + 3 coordinated agents, "clear step forward from Sonnet 4.5."
