Grok 4.20 是 xAI 的代码导向模型，支持 Thinking 模式，默认 200k 上下文，长上下文模式可达 2M token，速度适合迭代 Agent 工作流。

**优势**：最高 2M token 上下文，适合大型 repo、长转录文本和多文件任务；支持 Thinking 模式处理复杂编程和调试；速度适合 Cursor 迭代 Agent 工作流。

**计费**：API 用量池；基础价 $2/1M input，$0.20/1M cached，$6/1M output（200k 上下文内）；超过 200k 上下文后定价翻倍。
