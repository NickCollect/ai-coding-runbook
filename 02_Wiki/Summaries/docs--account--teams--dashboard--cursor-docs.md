Cursor web dashboard 是团队管理的中枢，涵盖计费、成员管理、使用分析、集成配置和安全设置。

**主要功能区**：Overview（团队活动概览）、Settings（隐私、计费、SSO、API Keys 等）、Members（成员管理和权限）、Analytics（使用数据）、Audit Log（安全事件记录，Enterprise 专属）、Integrations（版本控制和第三方工具连接）、Cloud Agents（运行状态和日志）、Bugbot、Usage（AI 请求和资源消耗）、Billing & Invoices。

**Teams/Enterprise 通用设置**：Privacy Settings（零数据保留政策）、Usage-Based Pricing（月度花费上限）、Team Marketplaces（插件分发）、SAML/OIDC SSO。

**Enterprise 专属设置**：Model Access Control（模型访问控制）、Enhanced Spend Limits（成员级别费用上限）、Auto Run Configuration（命令自动执行策略）、Repository Blocklist（禁止访问的仓库）、MCP Configuration、CursorIgnore 配置、.cursor 目录保护、AI Code Tracking API、Audit Log（SCIM）。
