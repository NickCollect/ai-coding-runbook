Standard **Contributor Covenant 2.0** Code of Conduct for the MCP main spec repo. Substantively identical to the CoC used in the docs repo and other MCP repositories — covers harassment-free pledge, positive vs. unacceptable behaviors, enforcement responsibilities of community leaders, scope (community spaces + official representation), and the four-step enforcement ladder (Correction → Warning → Temporary Ban → Permanent Ban).

**Enforcement contact**: `mcp-coc@anthropic.com`. Adapted from Contributor Covenant 2.0 with Mozilla-inspired enforcement guidelines.
